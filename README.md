# 文档套打 · 飞书多维表格插件

将多维表格数据批量填充到 Word (.docx) 模板，支持打印和归档。

## 功能

- 上传 .docx 模板，自动识别 `{{变量名}}` 占位符
- 变量映射：绑定表格字段或手动输入固定文本（公司名称等固定内容）
- 转换规则：直接填充 / 数字大写 / 日期格式化 / 图片嵌入
- 图片变量：在模板中写 `{{%变量名}}`，自动下载附件图片嵌入文档
- 套打范围：当前记录 / 手动选择 / 全部记录
- 方案管理：保存、切换多套配置方案
- 打印：批量生成，合并到一次打印弹窗
- 归档：生成的 .docx 文件写回到指定附件字段
- 支持中/英/日三语，自动跟随飞书语言设置
